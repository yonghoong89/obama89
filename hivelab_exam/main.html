    first-child

    javascript

    • .firstChild : element와 text와 주석을 모두 인식하여 가장 앞에 있는것을 반환한다.
    • .firstElementChild : 첫번째 element를 반환한다. (IE9부터 지원)

    jQuery

    • $('.class:first') : 선택자의 첫번째 element를 반환한다.
    • $('.class:first-child') : 선택자의 부모를 기준으로 첫번째 element 모두를 반환한다.
    • $('tagName').first() : 선택자의 첫번째 element를 반환한다.

    last-child

    javascript

    • .lastChild : element와 text와 주석을 모두 인식하여 가장 마지막 있는것을 반환한다.
    • .lastElementChild : 마지막 element를 반환한다. (IE9부터 지원)

    jQuery

    • $('.class:last') : 선택자의 가장 마지막 element를 반환한다.
    • $('.class:last-child') : 선택자의 부모를 기준으로 마지막 element 모두를 반환한다.
    • $('tagName').last() : 선택자의 마지막 element를 반환한다.

    previousSibling

    javascript

    • .previousSibling : element와 text와 주석을 모두 인식하여 선택자의 형제 요소중 바로 앞에 element를 반환한다.
    • .previousElementSibling : 선택자의 형제 요소중 바로 앞에 element를 반환한다. (IE9부터 지원)

    jQuery

    • $('.class').prev() : 선택자의 형제 요소중 앞에 element를 반환한다.
    • $('.class').prevAll() : 선택자의 형제 요소중 앞에 element를 모두 반환한다.
    • $('.class').prevUntil('selector') : .class부터 selector사이에 element를 모두 반환한다.

    nextSibling

    javascript

    • .nextSibling : element와 text와 주석을 모두 인식하여 선택자의 형제 요소중 바로 다음 element를 반환한다.
    • .nextElementSibling : 선택자의 형제 요소중 바로 다음 element를 반환한다. (IE9부터 지원)

    jQuery

    • $('.class').next() : 선택자의 형제 요소중 다음 element를 반환한다.
    • $('.class').nextAll() : 선택자의 형제 요소중 다음 element를 모두 반환한다.
    • $('.class').nextUntil('selector') : .class부터 selector사이에 element를 모두 반환한다.

    javascript querySelector() / querySelectorAll()

    • javascript도 jquery처럼 괄호안에 css 선택자와 동일하게 넣어 작성할 수 있다.
      예) document.querySelector('.class') / document.querySelectorAll('.class')
    • querySelector() : 괄호안에 명시된 선택자가 여러개라도 그중 가장 앞에 있는 것만 선택된다. (IE8부터 지원)
    • querySelectorAll() : 괄호안에 명시된 선택자가 모두 선택된다. 다수의 Element가 선택되므로 Index번호를 통해 접근이 가능하다. (IE9부터 지원)

    prototype 정리

    - 모든 javascript Object(Array, String, Object, Number, Function, RegExp, Date 등)는 prototype을 가지고 있다.

    - 모든 javascript Object들은 Object.prototype으로 부터 상속받으며 Object.prototype은 prototype chain에서 최상위에 위치한다.

    - 모든 프로퍼티와 메서드들은 상속받은 부모의 prototype에 따라 정해진 규칙에 의해 사용할 수 있다.

    - 자신만의 prototype도 생성 할 수 있으며, 수정도 가능하다. 하지만, 표준 prototype은 수정할 수 없다.

    Aarry.prototype

    Arrary.join()

    • 배열요소를 문자열로 결합하고, 문자열로 반환한다.
    • 괄호안에 문자를 기준으로 배열요소를 결합하는데, 기본값은 ',' 이다.
                    arr = ['ab','cd','ef','gh'];
                arr.join(); //"ab,cd,ef,gh";
                arr.join(''); //"abcdefgh";
                arr.join('  '); //"ab  cd  ef  gh"; * 공백도 인식해서 결합함. *
                arr.join('-'); //"ab-cd-ef-gh";

    Arrary.reverse()

    • 배열요소를 순서를 뒤집어서 배열요소를 반환한다.
                    arr = ['ab','cd','ef','gh'];
                arr.reverse(); //['gh','ef','cd','ab'];

    Arrary.sort()

    • 배열요소를 오름차순으로 정렬하여 배열요소를 반환한다.
    • 알파벳(대,소문자), 가나다등 문자열 모두 고유의 번호를 가지고 있어 그 숫자에 맞춰 정렬된다.
    • 배열에 숫자를 넣어 정렬할 경우 숫자들을 모두 문자열로 인식하여 정렬한다.
    • 숫자를 오름차순과 내림차순으로 정확히 정렬하려면 정해진 공식을 넣어주면 된다.
                    arr = ["Banana", "Orange", "Apple", "Mango"];
                arr2 = ["감사", "해요", "사랑", "입니다"];
                arr3 = [25, 1000, 68278, 70];
                arr.sort(); // ["Apple", "Banana", "Mango", "Orange"]
                arr2.sort(); // ["감사", "사랑", "입니다", "해요"]
                arr3.sort(); // [1000, 25, 68278, 70] * 숫자를 문자로 인식하여 정렬 하므로 25보다 1000이 먼저 정렬된다.
                arr3.sort(function(a, b){return a-b}) // 오름차순 숫자 정렬 [25, 70, 1000, 68278]
                arr3.sort(function(a, b){return b-a}) // 내림차순 숫자 정렬 [68278, 1000, 70, 25]

    Arrary.concat()

    • 2개 혹은 그 이상의 배열요소를 결합하여 배열요소로 반환한다.
    • 괄호안에 결합할 배열인자들을 1개부터 여러개를 넣어 결합이 가능하며, 인자값 순서대로 배열들이 결합된다.
                    arr = ["Banana", "Orange", "Apple", "Mango"];
                arr2 = ["red", "yellow", "blue", "green"];
                arr3 = ["Cecilie", "Lone", "Tobias", "Linus"];
                arr.concat(arr2, arr3); // ["Banana", "Orange", "Apple", "Mango", "red", "yellow", "blue", "green", "Cecilie", "Lone", "Tobias", "Linus"]

    Arrary.slice()

    • 배열요소를 원하는 요소만 골라내서 새로운 배열요소를 반환한다.
    • 배열요소는 0부터 시작하며, -값도 넣을 수 있다.
    • 괄호안에 인자값(start, end)을 숫자 2개까지 넣을 수 있으며, start값은 포함하고, end값은 포함하지 않고 반환한다.
    • 괄호안에 인자값을 1개만 넣을 경우, 해당 인자값 이후의 모든 배열요소를 반환한다.
                    arr = ["Banana", "Orange", "Lemon", "Apple", "Mango"];
                arr.slice(2); // ["Lemon", "Apple", "Mango"]
                arr.slice(1,4); // ["Orange", "Lemon", "Apple"]
                arr.slice(-3,-1); // ["Lemon", "Apple"]

    Arrary.splice()

    • 배열요소를 지우거나, 추가해서 배열의 내용을 변경하고, ****삭제된 요소들의 배열을 반환한다. 아무것도 삭제되지 않았을 경우 빈 배열을 반환한다.
    • 괄호안에 인자값(index, howmany, item1, ....., itemX)을 넣어 사용한다.
    • 인자값을 item을 넣지 않으면 index에 해당하는 배열 부터 다음값 갯수만큼 삭제된다.
    • 인자값을 index만 넣을 경우, index에 해당하는 배열부터 이후 요소들은 모두 삭제된다. 
                    arr = ["Banana", "Orange", "Lemon", "Apple", "Mango"];
                arr.splice(2, 1, "kiwi", "cherry"); // ["Lemon"] *삭제된 2번인덱스 배열 반환
                arr // ["Banana", "Orange", "kiwi", "cherry", "Apple", "Mango"] arr배열의 내용이 변경됨.

    Arrary.push()

    • 괄호안에 값을 넣으면 마지막에 배열요소를 추가 하고, 배열길이 값을 반환한다.
                    arr = ["Banana", "Orange", "Lemon", "Apple", "Mango"];
                arr.push("kiwi"); // 6 (배열길이 반환)
                arr // ["Banana", "Orange", "Lemon", "Apple", "Mango", "kiwi"] arr배열의 내용이 변경됨.

    Arrary.pop()

    • 마지막 배열요소를 삭제하고, 삭제된 요소(해당요소가 문자이면 문자열, 숫자이면 숫자열)를 반환한다.
                    arr = ["Banana", "Orange", "Lemon", "Apple", "Mango"];
                arr2 = ["Banana", "Orange", "Lemon", "Apple", 555];
                arr.pop(); // "Mango"
                arr2.pop(); // 555
                arr // ["Banana", "Orange", "Lemon", "Apple"] arr배열의 내용이 변경됨.
                arr2 // ["Banana", "Orange", "Lemon", "Apple"] arr2배열의 내용이 변경됨.

    Arrary.shift()

    • 맨앞에 배열요소를 삭제하고, 삭제된 요소(해당요소가 문자이면 문자열, 숫자이면 숫자열)를 반환한다.
                    arr = ["Banana", "Orange", "Lemon", "Apple", "Mango"];
                arr2 = [555, "Orange", "Lemon", "Apple", "Mango"];
                arr.shift(); // "Banana"
                arr2.shift(); // 555
                arr // ["Orange", "Lemon", "Apple", "Mango"] arr배열의 내용이 변경됨.
                arr2 // ["Orange", "Lemon", "Apple", "Mango"] arr2배열의 내용이 변경됨.

    Arrary.unshift()

    • 괄호안에 값을 넣으면 맨앞에 배열요소를 추가 하고, 배열길이 값을 반환한다.
                    arr = ["Banana", "Orange", "Lemon", "Apple", "Mango"];
                arr.unshift("kiwi","pineapple"); // 7 (배열길이 반환)
                arr // ["kiwi", "pineapple", "Banana", "Orange", "Lemon", "Apple", "Mango"] arr배열의 내용이 변경됨.

    Arrary.toString()

    • 배열요소를 콤마(,)로 구분하여 문자열로 반환한다.
                    arr = ["Banana", "Orange", "Lemon", "Apple", "Mango"];
                arr.toString(); // "Banana,Orange,Lemon,Apple,Mango"

    Arrary.indexOf()

    • 인자값에 해당되는 인덱스값을 반환한다.
    • 해당되는 값이 없는 경우, -1 반환한다.
    • IE9부터 지원
                    arr = ["Banana", "Orange", "Lemon", "Apple", "Mango"];
                arr2 = ["Banana", "Orange", "Apple", "Mango", "Banana", "Orange", "Apple", "Mango"];
                arr.indexOf("Apple"); // 3
                arr2.indexOf("Apple", 4); // 6 ** "Apple"라는 문자를 4번째 인덱스 부터 찾음. ( lastIndexOf()를 쓰면 뒤에서 부터 찾을 수 있음)

    String.prototype

    String.replace()

    • 2개의 인자값(searchValue, newValue)이 필요하며, searchValue값을 newValue값으로 문자열을 변경할 수 있다.
    • newValue값에 숫자를 넣어도 문자열로 반환된다.
    • 정규표현식으로 값을 찾을 수 있다.
                    str = "Delicious Bananas";
                str.replace("Bananas", "Apples"); // "Delicious Apples"

    String.slice()

    • 문자열을 원하는 대로 잘라서 문자열을 반환한다.
    • 0부터 시작하며, -값도 넣을 수 있다.
    • 괄호안에 인자값(start, end)을 숫자 2개까지 넣을 수 있으며, start값은 포함하고, end값은 포함하지 않고 반환한다.
    • 괄호안에 인자값을 1개만 넣을 경우, 해당 인자값 이후의 모든 문자열 반환한다.
    • 공백도 포함된다.
                    str = "Delicious Bananas";
                str.slice(3, 15); // "icious Banan"
                str.slice(-1); // "s"

    String.split()

    • 문자열을 인자값(separator, limit)을 기준으로 배열로 반환한다.
    • 기준이 되는 separator값은 삭제된다.
    • limit값만큼 배열길이가 반환된다.
    • 공백도 포함된다.
                    str = "Delicious Bananas";
                str.split(); // ["Delicious Bananas"] 인자값이 없는경우 배열로만 반환
                str.split("i", 2) // ["Del", "c"] 문자열 "i"를 기준으로 배열길이 2만큼만 반환

    String.search()

    • 인자값에 해당하는 인덱스 값을 반환한다.
    • 인자값이 없는 경우 -1을 반환한다.
    • 문자열과 정규표현식을 사용할 수 있다.
    • 공백도 포함된다.
                    str = "Delicious Bananas";
                str.search("Ban"); // 10
                str.search("apple"); // -1

    String.match()

    • 인자값에는 정규표현식만 넣을 수 있으며, 매칭되는 문자들은 배열로 반환한다.
                    str = "The rain in SPAIN stays mainly in the plain"; 
                str.match(/ain/g); //["ain", "ain", "ain"]
                str.match(/ain/gi); //["ain", "AIN", "ain", "ain"]

    String.trim()

    • 인자값으로 받은 문자열의 양쪽 여백을 제거하여 반환한다.
    • IE9부터 지원
                    str = "    Hello World!    ";
                str.trim(); // "Hello World!" * 아래 기존 문자열과 비교!
                str; // "    Hello World!    " ** 문자열은 공백을 모두 포함한다.

    String.indexOf()

    • 인자값에 해당되는 인덱스값을 반환한다.
    • 해당되는 값이 없는 경우, -1 반환한다.
                    str = "Hello world, welcome to the universe.";
                str.indexOf("welcome") // 13
                str.indexOf("e", 5) // 14 ** "e"라는 문자를 5번째 인덱스 부터 찾음. ( lastIndexOf()를 쓰면 뒤에서 부터 찾을 수 있음)
