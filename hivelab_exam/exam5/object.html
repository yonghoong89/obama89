Q. 함수 호출시 new 를 붙인 것과 안 붙인 것의 차이를 설명하세요.

예시)
var obj = function () {
   console.log(this);
};
obj();
// new obj();

- obj(); --> 함수 호출 규칙에 따른 호출방법이며, 해당 함수의 this는 전역객체(window)가 바인딩된다.

- 기존 함수에 new연산자를 붙여서 호출 하면 해당 함수는 생성자 함수로 동작한다.

- 함수가 생성자로 호출되면, 함수 코드가 실행되기 전에 빈 객체가 생성된다.

- 빈 객체는 생성자 함수가 새로 생성하는 객체이며, 이 객체는 생성자 함수의 this로 바인딩 된다.

Q. 객체 내에서의 this 를 설명하세요.

예시)
var obj = {
   init : function () {
      console.log(this);
   }
};
obj.init();

객체의 프로퍼티가 함수일 경우, 이 함수를 메서드라고 부른다.

메서드를 호출할 때, 메서드 내부 코드에서 사용된 this는 해당 메서드를 호출한 객체로 바인딩 된다.

Q. 객체내의 모든 속성들의 값을 배열에 저장한후, abc 순으로 sorting 하여, 'abcdefg' 문자열로 출력해보세요.

Q. 'bcaebe' 문자열을 hasOwnProperty 를 사용해서, {a : 1, b : 2, c : 1, e : 2} 로 변경해보세요.

var obj = {};
var arr = "bcaebe".split('').sort();
for(var i = 0; i< arr.length; i++){
  if(obj.hasOwnProperty(arr[i])){
    obj[arr[i]] = obj[arr[i]] + 1;
  }else{
    obj[ arr[i] ] = 1;
  }
}
console.log(obj)
